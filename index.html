CSS Cheatsheet

Property Values Used with Elements
color #hex, rgb(), keyword any element that contains text
text-align left, right, center, justify block elements h1-h6, p, li, etc
text-decoration none, underline, overline, line-through, blink, inherit mostly with link (a/anchor) elements
text-transformation none, capitalize, uppercase, lowercase any element that contains text
line-height %, px block elements h1-h6, p, li, etc
letter-spacing normal, px any element that contains text
font-family font or font-family any element that contains text
font-size px, em any element that contains text
font-style normal, italic, oblique any element that contains text
font-weight normal, bold, 100-900 any element that contains text
background-color #hex, rgb(), keyword any element with a background
background-image url("") mostly with body
